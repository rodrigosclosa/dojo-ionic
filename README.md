# dojo-ionic

Install all modules and needed tools

```bash
npm install -g npm
npm install -g yo
npm install -g grunt
npm install -g grunt-cli
npm install -g bower
npm install
bower install
```


Distribute and build
```bash
gulp
```


Quick run
```bash
grunt serve
```


Add route
```bash
yo angular:route myroute
```


Add components bower
```bash
bower install bootstrap
```


Testing
```bash
grunt test
```
